# Cinema-Ticketing:-
Here is my semester final project. In this cinema ticketing system on ASP.NET having modules like:-
* Add/Remove Cinemas
* Add/Remove Movies
* Discounts 
* Book/Delete Ticket
* Change Cinema
* Food
* Contacts
* Rating
* Payment Section
* Watch History
In this project, we developed two sections, one is Admin side and other is Customer side.
